SportSee

An app to manage sports statistics. Created with React App and Recharts

Prerequisites

  • NodeJS v12.22.12
  • yarn v1.22.17 (or npm)

Launch JsDoc

- run the project with the backend or with the mocked datas

- npm run doc

JsDoc is now rendered at URL http://localhost:5500

Dependencies

  • React v18.0.0
  • React-dom v18.0.0
  • React-router-dom v6.3.0
  • Recharts v2.1.9
  • Prettier v2.6.0
  • Babel v7.17.2
  • EsLint v8.9.0
  • Axios v0.26.1
  • JSDoc v3.6.10

1 | Back-end

  1. Download or clone the back-end's repository :
git clone https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard.git
  1. Install dependencies :
# Yarn
yarn install
  1. Launch back-end on port 3000 (default port) :
# Yarn
yarn dev

2 | Front-end

  1. Clone this repository :

git clone https://github.com/MarineHgns/sport-see.git

  1. Install dependencies:
# Yarn
yarn install
  1. Launch dev server on port 3000 :
# Yarn
yarn dev
  1. Front-end is now rendered at URL http://localhost:3001 and you can navigate on the site with backend data or mocked data (like you want).

You can also skip the backend part and launch the project directly with the frontend part (mocked data only).
